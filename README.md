# Work With Eli — Freelance Portfolio

Premium Website Designer & Front-End Developer. Portfolio built with Next.js 15, React 19, TypeScript, Tailwind CSS, Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

## Portrait Placeholder

The portrait shown throughout the site is a stylized SVG monogram. To use a real photo:

1. Drop your photo into `/public/portrait.jpg`
2. Open `lib/constants.ts` and change `PORTRAIT.src` from `null` to `"/portrait.jpg"`
3. Redeploy

## Deploy

Push to GitHub, then import at vercel.com/new.
